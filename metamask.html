You're on the wrong Ethereum network.

While in beta, msgblocks only works on the Ropsten network, which doesn't appear to be network currently selected in MetaMask. Change it from the MetaMask's settings and try again.

From the MetaMask website:
MetaMask is a bridge that allows you to visit the distributed web of tomorrow in your browser today. It allows you to run Ethereum dApps right in your browser without running a full Ethereum node.

MetaMask includes a secure identity vault, providing a user interface to manage your identities on different sites and sign blockchain transactions.

Here's a video that explains how to install and configure it, and how it's used to make transactions with a sample app. The way it works on msgblocks is basically the same!
