Frequently Asked Questions

What do I need to use this service?

A web browser (either Chrome or Firefox) with the MetaMask extension installed + an Ethereum wallet address and some ETH amount to pay for the transaction.

How much does it cost to write a message on the blockchain?

It depends on the length of the text that you write;
longer text = more "gas" needed to run the transaction on the blockchain = more ETH to spend.

Encrypting a message with a password can also have an impact on the length.

Upon clicking the button to submit your message, a MetaMask dialog will popup, which will show you all these details, so that there are no surprises!

How much does it cost to submit a reply to a message?

Same as for writing a normal message (i.e. not a reply). It depends on how long it is.

Do you earn anything?

We apply a fee of 0.005 ETH per message.

Can I delete a message or reverse a transaction?

No, that's the nature of the blockchain. Once something is done, is done (which is also it's beauty, because your message will be permanent). Please refer to the T&C to know more.
